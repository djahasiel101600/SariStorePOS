# POS Keyboard Shortcuts Guide

This document describes all keyboard shortcuts available in the POS system for efficient, mouse-free operation.

## üîç Search & Navigation

| Shortcut   | Action             | Description                                           |
| ---------- | ------------------ | ----------------------------------------------------- |
| `Ctrl + K` | Focus search       | Jump to the product search bar instantly              |
| `/`        | Focus search (alt) | Alternative shortcut to focus search                  |
| `Enter`    | Add to cart        | When search is focused, adds the first result to cart |
| `Esc`      | Clear/Close        | Clears search text or closes open dialogs             |

| `F3`     | ‚Äî           | Card payment        | Switch to card payment                                 |

## üí° Best Practices for Keyboard-Only Workflow

### Typical Transaction Flow (No Mouse Required):

1. **Start Transaction**

   - Press `Ctrl + K` or `/` to focus search
   - Type product name or SKU
   - Press `Enter` to add first result to cart

2. **Add More Items**

   - Search automatically clears after adding item
   - Repeat step 1 for additional products

3. **Adjust Pricing (Variable Products)**

   - Use `Tab` to navigate to "Set Price" field
   - Enter custom price
   - Press `Enter` to confirm

4. **Select Payment Method**

   - Press `F1` for Cash (or `Alt + 1`)
   - Press `F2` for Mobile/GCash (or `Alt + 2`)
   - Press `F3` for Card
   - Press `F4` for Utang (or `Alt + 3`)
   - For Utang: `F9` or `Ctrl + U` to add/select customer
   - For Cash: `F6` to quickly jump to amount tendered field

5. **Complete Transaction**
   - For Cash: Enter amount tendered using keyboard
   - Press `Ctrl + Enter` to checkout

### Quick Actions:

- **Cancel/Clear**: Press `Esc` to cancel current action or clear search
- **Fresh Start**: `Ctrl + Shift + C` to clear cart and start over
- **Look Up History**: `Ctrl + H` to view recent sales
- **Need Help**: Press `?` or `Ctrl + /` anytime to see shortcuts

## üéØ Accessibility Features

- **No Mouse Required**: Every POS function is accessible via keyboard
- **Visual Hints**: Shortcuts are displayed on relevant buttons
- **Consistent Patterns**:
  - `F1-F4` for payment methods (primary access)
  - `F8-F10` for tools (scanner, customer, history)
  - `Ctrl` for actions and tools (alternative access)
  - `Alt` for mode switching (alternative for payment methods)
  - `Esc` for cancellation/closing
  - `/` and `?` for search and help (common patterns)

## üì± Mobile vs Desktop

Keyboard shortcuts are **automatically disabled on mobile devices** to prevent conflicts with mobile keyboards and touch interactions. On mobile, use the touch-optimized interface.

## üöÄ Tips for Maximum Efficiency

1. **Learn the Flow**: Memorize the transaction flow shortcuts first
2. **Use F-Keys for Speed**: F1-F4 for payment, F8-F10 for tools - fastest single-key access
3. **Laptop Users**: If your laptop requires Fn key, use Alt+1-3 alternatives for payment methods
4. **Visual Hints**: Payment method buttons show F-key shortcuts in top-right corner
5. **Practice Common Actions**: Focus on `Ctrl+K`, `Enter`, `F1-F4`, and `Ctrl+Enter`
6. **Keep Help Handy**: Press `?` anytime to reference all shortcuts
7. **Touch Type**: Once memorized, you can complete transactions without looking at keyboard
